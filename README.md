# AnkiNinja

Консольное приложение для изучения карточек из файловой структуры `cards/`.

## Формат файлов

Корневой каталог: `cards`

Пример содержимого файла:

```text
front | back
front | back
front | back
```

Можно раскладывать колоды по подпапкам и проваливаться в них рекурсивно.

## Запуск

```bash
python3 -m ankininja
```

Если папка с карточками лежит не в `./cards`:

```bash
python3 -m ankininja --cards-root /path/to/cards
```

## Управление

- `↑` / `↓` выбрать файл или папку
- `Enter` открыть
- `←` или `Backspace` назад
- `q` выход

В режиме изучения:

- карточки берутся в случайном порядке
- `Enter` показывает ответ
- после ответа `←` / `→` выбирают сложность
- `Enter` подтверждает сложность
- уровни сложности подсвечены цветом: от светло-зеленого к темно-красному
- `Ctrl+B` назад к списку
- `Ctrl+X` выход
- обычная печать всегда пишет в заметки
- `Ctrl+A` очищает заметки целиком
- `Backspace` удаляет символ
- `Ctrl+N` добавляет новую строку в заметках

Заметки сохраняются отдельно и не меняют файлы колод.

## Логика повторения

- сессия продолжается, пока все карточки не будут отмечены как `Легко`;
- `Очень сложно` возвращается почти сразу, через 1 карточку;
- `Сложно` возвращается через 3 карточки;
- `Нормально` возвращается через 6 карточек;
- `Легко` убирает карточку из текущей сессии.

## Статистика

Для файлов показываются:

- количество карточек;
- сколько раз колода открывалась для изучения;
- когда колода открывалась в последний раз.

Статистика хранится в `~/.ankininja/study_state.json`.
